تعلم HTML • CSS • JavaScript

صفحة واحدة. كتابة كود مباشرة. مشاريع حقيقية.

HTML – الهيكل

اكتب HTML أدناه → شاهد النتيجة على اليمين.

نصيحة: بعد هذا الدرس، اصنع صفحة سيرة ذاتية – العملاء يدفعون 200–600 ريال!

CSS – التنسيق

عدّل الـ CSS → شاهد التغيير فورًا.

اكسب المال: اعرض إعادة تصميم صفحة هبوط على فايفر – 150–400 ريال لكل طلب.

JavaScript – التفاعل

اكتب JS → شاهد التنفيذ فورًا.

اكسب المال: اصنع آلة حاسبة تفاعلية (مؤشر كتلة الجسم، القروض) – بعها على CodeCanyon.

اصنع واكسب – مشاريع صغيرة

  1. موقع شخصي – HTML + CSS (يوم واحد)
  2. قائمة مهام – HTML + CSS + JS (يومان)
  3. ويدجت الطقس – JS + API مجاني (3 أيام)

ضعها على GitHub → شارك الرابط → احصل على طلبات!

الخطوة التالية: بعد أي مشروع، انشر في Upwork أو مستقل.
